# Sistema-de-Control-para-Planta-Piloto-de-4-Estanques
En este repositorio se documentará el diseño e implemntación de control automático para una planta piloto de 4 estanques.  Esto incluye control en tiempo real, protocolos de comunicación, instrumentación, diseño de HMI bajo a norma ISA-101 y control predictivo
